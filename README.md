WinStats
WinStats is a lightweight, real-time hardware monitoring tool built with C# and .NET 8. Designed for efficiency, it provides essential system performance metrics in a clean, intuitive interface.

Features
Real-time Monitoring: Track hardware performance metrics as they happen.

Lightweight: Minimal resource footprint, built for speed and reliability.

Intuitive UI: Simple, user-friendly interface for at-a-glance monitoring.

Installation
Navigate to the Releases page.

Download the latest WinStats_Setup.exe.

Run the installer and follow the on-screen instructions to complete the setup.

Launch the application from your desktop shortcut.

Built With
Language: C#

Framework: .NET 8

Installer: Inno Setup
